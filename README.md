## Daily View

<hr />

### start backend server

create .env file from .env.example

#### setup postgres

- DB_PASSWORD=password
- DB_USERNAME=user
- DB_NAME=dailyview
- DB_PORT=5432
- DB_HOST=localhost

```
npm i && npm run start:dev-hot
```

#### api swagger docs
